# 시스템 정의서 (System Definition Document)

## 1. 프로젝트 명칭
**MiniDrive: 조직 내 협업 가속화를 위한 클라우드 기반 통합 파일 관리 시스템**

## 2. 개발 시스템 명칭
**Cloud-Native File Management System**

## 3. 개발 시스템 설명 (System Overview)
이 시스템은 프로젝트 기반의 협업 환경의 파일 공유 과정에서 발생할 수 있는 버전 관리의 어려움과 분산된 파일등을 정리하고 접근하게 쉽도록 관리할 수 있도록 하는 중앙관리 기반의 클라우드 공유 시스템이다.
약 200명의 조직원들이 동시해 접속하여 파일을 안정적으로 업로드, 다운로드, 공유할 수 있는 환경을 제공하며 외부 협력 업체와의 안전한 링크 공유 및 파일 버전을 기록해 복원할 수 있는 기능을 통해 데이터의 보안성을 보장한다.

## 4. 핵심 기능 (Key Features)

1. **중앙 집중식 파일 저장 및 관리 (Centralized Storage)**
   - 다양한 종류의 업무 파일(문서, 이미지, PDF, 설계 파일 등)을 크기 제한 없이 업로드 및 다운로드
   - 파일 업로드 시 파일명, 날짜, 크기 등 메타데이터 자동 기록 및 관리 

2. **다단계 폴더 구조 및 체계화 (Folder Organization)**
   - 프로젝트별 다단계 폴더 생성, 이름 변경, 삭제 및 폴더 간 파일 이동 기능 지원 
   - 실수로 삭제한 파일을 일정 기간 내에 되살릴 수 있는 휴지통(복구) 기능 제공 
3. **세분화된 파일 공유/권한 제어 (File Sharing & Permissions)**
   - 사용자별 보기, 수정, 댓글 작성 권한을 세부적으로 설정하여 내부 공유 지원 
   - 외부 사용자를 위한 만료 기간 설정 가능 공유 링크 생성 기능 
4. **다중 조건 검색 엔진 (Fast Search Engine)**
   - 파일 이름 기반 기본 검색 및 업로드 날짜, 업로더, 파일 유형별 상세 필터링 지원
   - 200명 동시 접속 환경에서도 지연 없는 빠른 검색 결과 표시 
5. **히스토리 기반 버전 관리 (Version Management)**
   - 일 수정 시 이전 버전을 자동으로 보관하여 버전 목록 확인 및 특정 시점 복원 기능 제공
6. **관리자 전용 계정 및 보안 관리 (Administration & Security)**
   - 이메일 기반 계정 생성/삭제 및 사용자별 저장 공간 할당량 관리 
   - 로그인 인증, 권한 기반 접근 제어(RBAC), 데이터 전송 시 암호화 적용

| **타겟 사용자** | 내부 조직 및 외부 협력사 | 개인 및 팀 단위 문서 협업 | 범용 클라우드 사용자 |

## 6. 향후 확장 가능성 (Future Possibilities)
* **문서 공동 편집:** 웹 상에서 여러 명의 사용자가 동시에 문서를 수정하는 모듈 도입
* **지능형 자동 분류:** AI를 활용하여 업로드된 파일을 성격에 따라 자동 폴더링
* **AI 기반 검색:** 파일의 내용을 분석하여 키워드만으로 관련 파일을 찾아주는 기능
